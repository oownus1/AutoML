# AutoML - Pycaret, Prophet, Pororo

```
NASA 데이터 분석 교육에서 주어지는 과제를 앞두고 상무님께서 AUTOML이라는 자동코딩 개념에 대해 알려주셨습니다.
데이터 분석 공부 과정에서 이 개념을 알면 적용하고 수행하는데 도움이 될 것이라고 말씀하셨습니다.
AUTOML 공부지도를 해주셔서 이와 관련된 공부를 기록합니다

데이터 분석가의 공부방향이 있다고 느낍니다.
파이썬 기초 -> 판다스 -> 머신러닝 공부 단계로 순차적으로 나아가는데 이 과정이 굉장히 길기도 하고 공부양이 방대합니다.
AUTOML이란 이 과정을 크게 하나로 통합하여 간단하게 수행할 수 있는 자동 머신러닝 개념을 의미합니다.

AUTOML 라이브러리 
PYCARET : 싸이킷런, 이상치 탐지, 해킹, 연관규칙(ex 자연어리뷰, 부정/긍정)
PROPHET : 페이스북에서 만든 시계열 데이터 분석해주는 라이브러리(주가 데이터,가격 데이터, 코인시세, 매출액) -> 앞으로의 가격, 변동성 예측
Pororo : 카카오에서 만든 한글전용 자연어 처리 분류기(자연어 생성, 분류)
```
```
회귀 분석과정 
1. setup을 한다
2. 모델을 돌린다
3. 결과가 나오면 모델을 선택한다.(n_select속성에서 선택하고 싶은 개수 선택can)
4. 이를 토대로 블렌딩이나 앙상블을 한다 -> dl 결과로 성능을 알 수 있다
5. 이를 통해 예측을 수행한다. (Label이 최종 예측값이 된다)
6. 시각화로 모델분석의 영향도 분석(어떤 컬럼들이 중요하게 작용을 했는지 판단can)
```
